# team-docs

Static documentation and published decks for the Consumer Risk / FNPL MBR workflow.

## GitHub Pages

After Pages is enabled (branch **main**, folder **/**), the site root is:

**https://namrataverma-rgb.github.io/team-docs/**

## FNPL MBR

Monthly HTML decks live under **`fnpl-mbr/`**, for example:

**https://namrataverma-rgb.github.io/team-docs/fnpl-mbr/FNPL_MBR_Apr26.html**

Add new months by committing `FNPL_MBR_<Mon><yy>.html` in `fnpl-mbr/` and pushing to `main`.

This layout mirrors internal team-docs style paths (`…/team-docs/fnpl-mbr/…`) on public GitHub Pages.
